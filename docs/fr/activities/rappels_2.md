# Rappels - variable par défaut

Vous travaillerez dans le script `exo_02.py` du package `exos` qui existe déjà et possède la 
fonction suivante :

```python
def add_knight(knight:str, kingdom:list=[]):
    kingdom.append(knight)
    return kingdom
```

Testez la fonction et identifiez ce qui ne va pas.

Proposez une solution pour éviter ce *bug*.

Dans le cas idéal, travaillez avec des tests.
